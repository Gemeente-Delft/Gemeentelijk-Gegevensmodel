# Homelessness

If clients are (or have been) homeless, this is recorded in *Homelessness*. This object type has a start and end date, so a *Client* may have multiple occurrences.

![Homelessness data model][dakloos]

<em>Diagram (in Dutch): data model for Homelessness.</em>

This is a relatively simple information model since it has a single object type. It records the period of homelessness, any consents, and the original municipality of the homeless person.
