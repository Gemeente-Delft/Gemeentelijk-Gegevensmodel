# Generic Heritage Object Types

Generic data model for Heritage.
The next figure shows the relationship of the *Prinsenhof* object types to the generic Heritage data definitions. A *Museum Object* is a *Heritage Object*, and photos and videos are linked to it. The figure also shows how historical persons are linked to heritage objects. Historical persons are derived from the object type *Natural Person* and so inherit attributes such as name, date of birth, date of death and gender.
