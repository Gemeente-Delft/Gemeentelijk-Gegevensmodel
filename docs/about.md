# Over het Gemeentelijk Gegevensmodel

Het Gemeentelijk Gegevensmodel (GGM) is ontwikkeld door de Gemeente Delft. Het bedoeld als bouwsteen in de ambitie om informatiegestuurd werken binnen de gemeente vorm te geven.

Het GGM is uitgewerkt en geschreven door <a href="https://www.linkedin.com/in/arjenbrienen/" rel="author me">Arjen Brienen</a> en <a href="https://www.linkedin.com/in/ashkanashkpour/" rel="author me">Ashkan Ashkpour</a> in opdracht van de Gemeente Delft.
