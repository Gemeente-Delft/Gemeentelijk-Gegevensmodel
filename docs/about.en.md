# About the Delft Municipal Data Model

The **Delft Municipal Data Model** (Dutch: *Gemeentelijk Gegevensmodel*, abbreviated **GGM**) was developed by the Municipality of Delft. The Dutch abbreviation *GGM* is retained throughout this site. It is intended as a building block for the ambition to give shape to data-driven working within the municipality.

The GGM was elaborated and written by <a href="https://www.linkedin.com/in/arjenbrienen/" rel="author me">Arjen Brienen</a> and <a href="https://www.linkedin.com/in/ashkanashkpour/" rel="author me">Ashkan Ashkpour</a> on commission of the Municipality of Delft.
